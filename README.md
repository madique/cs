# CS2 Config Documentation

Этот файл описывает все пользовательские бинды и настройки управления.

**Конфиги 1 и с одинаковые, кроме **приседания и ходьбы:**
В конфиге c.cfg `SHIFT` — это приседание, `CTRL` — это тихая ходьба.
В конфиге 1.cfg `CTRL` — это приседание, `SHIFT` — это тихая ходьба.

***

# Установка
* Нажать на иконку игры в стиме правой кнопкой -> просмотреть локальные файлы.
* Положить конфиг в /game/csgo/cfg
* В консоли написать: exec c

## ⚠️ Важные особенности
*   **Приседание и Ходьба:** В конфиге c.cfg `SHIFT` — это приседание, `CTRL` — это тихая ходьба.
*   **Прыжок:** Работает и на `Пробел`, и на `Колесико ВВЕРХ`.
*   **Drop Bomb:** Скрипт быстрого сброса всего инвентаря на клавишу `O`.

---

## 🏃 Передвижение
| Клавиша | Действие | Описание |
| :--- | :--- | :--- |
| `W` / `A` / `S` / `D` | Перемещение | Стандартное перемещение |
| `SHIFT` | **+duck** | **Приседание** (Обрати внимание!) |
| `CTRL` | **+sprint** | **Тихая ходьба** (Shift-walk) |
| `SPACE` | +jump | Прыжок |
| `MWHEELUP` | +jump | Прыжок (для баннихопа) |

## 💣 Гранаты (Быстрые слоты)
| Клавиша | Граната |
| :--- | :--- |
| `4` | Осколочная (HE Grenade) |
| `R` | Флешка (Flashbang) |
| `V` | Дым (Smoke) |
| `T` | Молотов / Зажигательная |
| `Z` или `0` | Декой (Decoy) |

## 🖱️ Мышь и Оружие
| Клавиша | Действие |
| :--- | :--- |
| `MOUSE1` | Стрельба |
| `MOUSE2` | Прицел / Альт. атака |
| `MOUSE3` (Колесо) | **Смена рук** (Левая/Правая) |
| `MOUSE4` (Боковая) | Осмотр оружия |
| `MOUSE5` (Боковая) | Пинг (Метка) |
| `MWHEELDOWN` | *Не назначено* |
| `Q` | Предыдущее оружие (Lastinv) |
| `F` | Перезарядка |
| `G` | Выбросить оружие |
| `O` | **Drop All** (Скрипт: выкидывает вообще всё из инвентаря) |

## 🛒 Закупка (Buy Binds)
| Клавиша | Что покупает |
| :--- | :--- |
| `B` | Открыть меню закупки |
| `F1` | AK-47 / M4A1 / M4A1-S |
| `F2` | AWP |
| `F3` | Desert Eagle |
| `F4` | Броня + Шлем (или просто Броня) |
| `F5` | **Фулл раскид:** 2 Флешки, Смока, Молотов, Дефуза |
| `F6` | **Альт раскид:** Флешка, Хаешка, Смока, Молотов, Дефуза |

## 🖥️ Интерфейс и Коммуникация
| Клавиша | Действие |
| :--- | :--- |
| `E` | Использовать + **Показать снаряжение команды** (X-Ray союзников) |
| `TAB` | Таблица очков |
| `CAPSLOCK` | **Зум радара** (Переключение масштаба карты) |
| `P` | Вкл/Выкл счетчик FPS |
| `~` (Ё) | Консоль |
| `C` | Голосовой чат |
| `X` | **Мут войс чата** (Вкл/Выкл голосовую связь) |
| `Y` | Чат (Общий) |
| `U` | Чат (Командный) |

## ⚙️ Сервер и Голосование
| Клавиша | Действие |
| :--- | :--- |
| `DEL` | Взять паузу (Callvote StartTimeOut) |
| `END` | Голосовать ЗА (F1 / Option 1) |
| `PGDN` | Голосовать ПРОТИВ (F2 / Option 2) |
| `KP_MULTIPLY` (*) | Написать в чат `!stay` |
| `KP_MINUS` (-) | Написать в чат `!switch` |

**Настройки звука:** Громкость `0.5`, громкость войс-чата `0.4`. Музыка в конце раунда и MVP отключена.
